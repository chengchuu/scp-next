A Modern TypeScript CLI and Library for Secure File Transfers

## Install

Use scp-next via [npm](https://www.npmjs.com/package/scp-next).

```bash
npm install scp-next --save
```

Of course, you can also download this file and serve it yourself. The file locates at the `lib/scp-next.min.js`.

## Usage

Foo foo foo example.

```javascript
import { foo } from "scp-next";

foo(); // true
```

## Contributing

### Development Environment

| Dependency | Version  |
|------------|----------|
| Node.js    | v22.21.1 |

### Scripts

Install Dependencies:

```bash
npm i
```

Development:

```bash
npm run dev
```

Build:

```bash
npm run build
```

Test:

```bash
npm run test
```

Documentation:

```bash
npm run docs
```

Docker:

```bash
docker compose up -d --build
```

Visit: <http://localhost:8080>
